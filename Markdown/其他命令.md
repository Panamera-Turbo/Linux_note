# 其他命令

*2020年7月8日*

| 命令  | 作用                                                         |
| ----- | ------------------------------------------------------------ |
| man   | 帮助文档  man "命令"   ：<br />可以查看命令的详情<br />man man可以查看首页 |
| echo  | 1.输出指定字符串 echo “xxx”<br />2.查看环境变量：echo \$"环境变量"<br />3.查看上一次程序退出值：echo $? |
| alias | 设置命令别名（临时）<br />alias  别名=命令  <br />若什么都不加，显示目前所有的别名设置 |
| file  | file 文件：查看文件格式                                      |
| ldd   | 查看依赖的所有动态库                                         |
| date  | 查看/修改时间                                                |
|       |                                                              |
