# 文件和目录的拷贝

*2020年2月19日9:15*

### cp命令。既可拷贝文件也可拷贝目录



| 对象 | 用法                                                         |
| ---- | ------------------------------------------------------------ |
| 文件 | cp 要拷贝的a（母本）待存放的b  <br />eg：cp a b<br/>若b不存在，则创建b并将啊中的内容复制到b里面<br>若b存在，覆盖b的内容 |
| 目录 | cp a b -r  <br />若b不存在，创建一个内容和a一样的b  <br />若b存在，将整个目录a放进b中，即 /b/a |
