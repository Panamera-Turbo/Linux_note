# 文件和目录属性的查看

*2020年2月20日*

- 常见：

  > od，wc，du，df

 

| 命令 | 适用于       | 详情                                                         |
| ---- | ------------ | ------------------------------------------------------------ |
| wc   | 文本文件     | 依次显示：行数   单词个数（以空格为准） 字节数  文件名    <br> -o 只显示字节数  -l只显示行数  -w只显示字数 |
| od   | 二进制文件   | -t参数 ： 指定显示格式。  具体参数如下： <br>c  ASCII字符  <br>f   浮点数  <br>d  有符号十进制  <br>o  八进制  <br>u  无符号十进制  <br>x  十六进制 |
| du   | 目录大小     | 显示当前目录及相邻的下一级目录大小，加参数 -h 才能看明白     |
| df   | 磁盘使用情况 | -h                                                           |
